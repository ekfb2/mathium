# Mathium

Mathium is a Unit 5 math quiz site with:
- Interactive quiz questions
- AI-powered hints
- Statistics and scatter plot practice
- A clean, mobile-friendly design

## Live site
https://mathium.onrender.com/quiz-500-generated.html

## Features
- Multiple choice, number, and text answer questions
- Instant feedback after each answer
- AI chat helper for math questions
- Unit 5 topics including:
  - Scatter plots
  - Correlation
  - Lines and curves of best fit
  - Interpolation and extrapolation
  - Mean, median, mode, range, quartiles, and box plots

## How it works
- Open the site in a browser
- Answer questions one by one
- Use the AI chat for help when needed

## Deployment
This project is deployed on Render as a Node.js web service.

## Files
- `server.js` - backend server
- `quiz-500-generated.html` - main quiz page
- `package.json` - dependencies and start script
